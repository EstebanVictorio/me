## Design Context

### Users
Recruiters, engineering managers, and potential collaborators viewing this as a professional portfolio. Primarily viewed during work hours on desktop, but mobile matters. The audience is technical — they'll notice code quality signals in the design itself.

### Brand Personality
Precise, cross-domain, understated. Esteban bridges wildly different industries (radiology → scooters → payments) with a consistent technical practice. The site should feel like the work: reliable, well-structured, no wasted motion.

### Aesthetic Direction
Editorial / technical document aesthetic. Typography-forward, light as the primary theme (portfolio is a professional artifact, meant for daytime viewing). Deep navy-tinted color palette that echoes the CV's existing color language. Dark mode is available as a user preference and feels like a clean developer tool environment, not a flashy dark app.

**Fonts:**
- Display / name: Unbounded (900 weight) — geometric, wide, architectural
- Body / everything else: Epilogue (300–700) — humanist, readable, slightly distinctive

**Colors (OKLCH):**
- Background: `oklch(97.5% 0.005 255)` light / `oklch(13.5% 0.022 258)` dark
- Accent (navy blue): `oklch(47% 0.215 258)` light / `oklch(68% 0.19 258)` dark
- Warm (amber): `oklch(58% 0.155 40)` light / `oklch(71% 0.145 40)` dark
- Company names use the warm amber; section labels and links use the navy accent

**Stack:** Astro 6 + Tailwind v4 + React 19 (for interactive islands) + Headless UI v2

### Design Principles
1. **Breadth as the story** — 7 companies, 6 industries. The cross-domain career is the differentiator; the design should surface it, not bury it.
2. **Typography does the work** — hierarchy comes from type scale and weight contrast, not from cards, shadows, or decorative elements.
3. **Precision over decoration** — thin rules, tabular numbers for dates, careful spacing. Nothing that isn't load-bearing.
4. **Confident understatement** — no gradient text, no glow effects, no hero animations that loop. One entrance animation, then stillness.
5. **Accessible throughout** — Headless UI components for all interactive elements; WCAG-compliant contrast in both themes.
